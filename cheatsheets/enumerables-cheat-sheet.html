Enumerable Methods

Enumerable is a Ruby module containing methods that sort, search, and traverse over each element of a collection of data. The Enumerable Ruby Docs lists all these handy methods for your viewing pleasure.

When passing a block to a hash element, the hash yields each key-value pair as an array in the format [key, value]. Thus while blocks acting on arrays are often described using element or e ({|e| puts e}, hashes commonly use |key, value| or |k,v|.

It's also worth keeping in mind that the order a hash yash yields its pairs can be unpredictable.

#all?

This method passes each element of the object to the block and returns true if the block never returns false. This is a good method to check if a collection of elements all have the same property.

    Notes:
  • #none? works opposite to#all?.
numbers = [1,2,3,4,5]
  numbers.all? {|num| num >=0 }
  => true

#each

The #each method is an effective replacement for a for loop. This method calls the block of code once for every element in self (in this case the array) and returns the original self object. See #map for a method to return a new value based on the block. 

array = ["This","is","a","demonstration"]array.each {|element| puts element}
  => This
  => is
  => a
  => demonstration

#each_cons vs #each_slice

#each_cons(n) iterates through each element in an object, returning the following n consecutive elements. It stops when it prints the last element.

#each_slice(n) performs a similar task, but slices n elements. 

array = ["~~~~", "Join,", "or", "die.", "~~~~"]
array.each_cons(2) {|element| print element}
  => ["Join", "us,"]["us,", "or"]["or", "die."]["die.", "~~~~"]
array.slice(2) {|element| print element}
  => "~~~~ Join,"
     "or die."
     "~~~~"

#inject

This method combines all elements of an our object (array) using the command inside the block. The first element in the block (sum) is called the accumulator value. Each element in the block (num) is passed to the accumulator. At the end of the iteration, the final value (sum) is returned.I think about inject by using it to sum numbers. The second element in the block alters the previous element. 

numbers = [1,2,3,4,5]
  numbers.inject {|sum,num| sum + num}
  => 15

#map

This method is similar to #each except it actually returns a new array containing the results of running the block on each element of the object passed through.

#collect is an alias for #map. 

numbers = [1,2,3,4,5]
  numbers.map {|num| num * 2}
  => [2, 4, 6, 8, 10]
  numbers.collect {|num| num * 2} == numbers.map {|num| num * 2}
  => true

#select

This method returns an array containing all elements for which the block evaluates as true. In the example below, #select searches each element of the array for a length >= 3. If this evaluates true, the element is included in the new array to be returned.

    Notes:
  • #find_all is an alias for #select.
  • #reject works opposite to #select, returning an array of values that evaluate as false.
array = ["This","is","a","demonstration"]
  array.select {|element| element.length >=3}
  => ["This", "demonstration"]
  array.reject {|element| element.length >=3}
  => ["is", "a"]
